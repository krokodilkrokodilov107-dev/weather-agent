# Weather Agent 🌤️

## Что делает этот агент?

**Weather Agent** — это умный помощник по погоде, который отвечает на ваши вопросы естественным языком и дает советы по одежде в зависимости от температуры. Агент получает актуальную информацию о погоде в вашем городе и может даже сравнивать условия в разных местах, чтобы помочь вам лучше спланировать день.

## Как использовать API

### Пример 1: Узнать погоду в городе

Отправьте вопрос о погоде в свой город:

```bash
curl -X POST http://localhost:5000/analyze \
  -H "Content-Type: application/json" \
  -d '{"question": "Какая сейчас погода в Москве?"}'
```

**Ответ:**
```json
{
  "question": "Какая сейчас погода в Москве?",
  "answer": "В Москве сейчас ясно, температура 22°C. Рекомендуется одеть рубашку или тонкий свитер."
}
```

### Пример 2: Сравнить погоду в разных городах

Если вы хотите узнать, где теплее, отправьте вопрос на сравнение:

```bash
curl -X POST http://localhost:5000/analyze \
  -H "Content-Type: application/json" \
  -d '{"question": "Сравни погоду в Москве, Санкт-Петербурге и Казани"}'
```

**Ответ:**
```json
{
  "question": "Сравни погоду в Москве, Санкт-Петербурге и Казани",
  "answer": "В Москве 22°C, в Санкт-Петербурге 18°C, в Казани 25°C. Самый теплый город — Казань, самый холодный — Санкт-Петербург. Разница температур составляет 7 градусов."
}
```

Также доступна проверка статуса сервиса:

```bash
curl http://localhost:5000/health
```

## Переменные окружения (настройка)

Для работы приложения необходимо установить две переменные окружения:

### 1. **ANTHROPIC_API_KEY**
- Ключ API от компании Anthropic (Claude AI)
- Где получить: https://console.anthropic.com
- Пример: `export ANTHROPIC_API_KEY="sk-ant-..."`

### 2. **OPENWEATHER_API_KEY**
- Ключ API от OpenWeatherMap (данные о погоде)
- Где получить: https://openweathermap.org/api
- Пример: `export OPENWEATHER_API_KEY="12abc34def56ghi7890..."`

### 3. **PORT** (опционально)
- Порт для запуска приложения (по умолчанию 5000)
- Пример: `export PORT=8080`

## Топ-3 проблемы и решения

### ❌ Проблема 1: "ANTHROPIC_API_KEY не установлена"

**Почему это происходит:** Приложение не может найти ключ API от Claude AI.

**Как решить:**
1. Получите ключ на https://console.anthropic.com
2. Установите переменную окружения:
   ```bash
   # На Linux/Mac
   export ANTHROPIC_API_KEY="ваш_ключ_здесь"
   
   # На Windows (PowerShell)
   $env:ANTHROPIC_API_KEY="ваш_ключ_здесь"
   ```
3. Перезапустите приложение

### ❌ Проблема 2: "OPENWEATHER_API_KEY не установлена"

**Почему это происходит:** Приложение не может получить данные о погоде без ключа OpenWeatherMap.

**Как решить:**
1. Зарегистрируйтесь на https://openweathermap.org/api
2. Получите свой ключ в личном кабинете
3. Установите переменную окружения:
   ```bash
   # На Linux/Mac
   export OPENWEATHER_API_KEY="ваш_ключ_здесь"
   
   # На Windows (PowerShell)
   $env:OPENWEATHER_API_KEY="ваш_ключ_здесь"
   ```
4. Перезапустите приложение

### ❌ Проблема 3: "Город не найден" или "Ошибка при получении данных о погоде"

**Почему это происходит:** Вы написали название города неправильно, город слишком маленький или нет доступа в интернет.

**Как решить:**
1. **Проверьте название города:** используйте английское название, например:
   - ❌ `Москва` → ✅ `Moscow`
   - ❌ `Санкт-Петербург` → ✅ `Saint Petersburg`
   - ❌ `Бишкек` → ✅ `Bishkek`
2. **Проверьте интернет:** убедитесь, что соединение с интернетом работает
3. **Попробуйте известный город:** сначала протестируйте с названием большого города, например "London" или "New York"
4. **Проверьте ключ API:** убедитесь, что OPENWEATHER_API_KEY верен

---

**Нужна помощь?** Проверьте, что оба ключа API правильно установлены и ваше интернет-соединение работает. Большинство проблем решаются именно так! 🚀
